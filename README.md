# Getting started

1. Fork this repository on GitHub.
2. Clone *your forked repository* with `git clone https://github.com/YOURUSERNAME/realtime-chat.git`
3. `cd realtime-chat`
4. Install dependencies and start:

```shell
# Install dependencies:
$ yarn

# Start projects:
$ yarn start
```
